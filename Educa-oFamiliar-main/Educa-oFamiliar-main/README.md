# Educa-oFamiliar
Projeto de app auxiliar na educacao familiar.

## EduHouse
Aplicativo feito com React Native + Expo e UI Kitten para incentivar criancas a cumprir tarefas e trocar pontos por recompensas.

Telas principais:
- Pontos: lista de tarefas e acumulador de pontos.
- Recompensas: lista de premios para trocar por pontos.

## Como rodar
Entre na pasta do app e inicie o Expo:

```bash
cd EduHouse
npm run start
```

## Publicar no Expo (OTA)
1. Faça login no Expo:

```bash
cd EduHouse
npx expo login
```

2. Publique a atualizacao:

```bash
cd EduHouse
npx expo publish
```

Depois disso, o Expo mostra o QR Code do build publicado para usar no Expo Go.
